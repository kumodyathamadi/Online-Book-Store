The Online Book Store is a simple web application developed to manage books efficiently through a user-friendly interface. This project allows users to view, add, update, and delete books, demonstrating core web development concepts and database integration.
The application is built using Spring Boot and follows the MVC architecture. Thymeleaf is used for server-side rendering, Bootstrap ensures a responsive and clean UI, JPA handles database operations, and MySQL is used as the relational database.
This project is ideal for students and beginners to understand CRUD operations, Spring Boot fundamentals, and full-stack web application development.
